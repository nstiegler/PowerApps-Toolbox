# Input

Basic text input field.

## Preview

A single-line text input with border, hint text, and focus states.

## Tags

`ui` `form` `input` `basic`

## Required Variables

None

## Setup

1. Copy contents of `input.yaml` and paste into your screen
2. Update `HintText` property with your placeholder text
3. Reference `Input.Text` to get the entered value

## Customization

- **Placeholder:** Change `HintText`
- **Border:** Modify `BorderColor` and `BorderThickness`
- **Focus state:** Adjust `FocusedBorderThickness` and `PressedBorderColor`
- **Size:** Change `Width` and `Height`
- **Font:** Modify `Font` and `Size` properties

## Dependencies

None
