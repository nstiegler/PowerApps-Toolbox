# Button

Basic filled button with rounded corners.

## Preview

A solid black button with white text. Includes hover and pressed states.

## Tags

`ui` `input` `basic`

## Required Variables

None

## Setup

1. Copy contents of `button.yaml` and paste into your screen
2. Update `Text` property with your label
3. Add your logic to `OnSelect`

## Customization

- **Colors:** Change `Fill` (background) and `Color` (text)
- **Size:** Adjust `Width` and `Height`
- **Corners:** Modify `RadiusBottomLeft`, `RadiusBottomRight`, `RadiusTopLeft`, `RadiusTopRight` (default: 5)
- **States:** Customize `HoverFill` and `PressedFill` for interaction feedback

## Dependencies

None
