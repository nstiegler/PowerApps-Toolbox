# Button Outline

Ghost/outline button variant with transparent fill.

## Preview

A button with transparent background and visible border. Text is dark. Fills with gray on hover.

## Tags

`ui` `input` `basic`

## Required Variables

None

## Setup

1. Copy contents of `button-outline.yaml` and paste into your screen
2. Update `Text` property with your label
3. Add your logic to `OnSelect`

## Customization

- **Border:** Change `BorderColor` and `BorderThickness`
- **Hover state:** Modify `HoverFill` (default: light gray)
- **Size:** Adjust `Width` and `Height`
- **Corners:** Modify radius properties (default: 5)

## Dependencies

None
